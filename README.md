# Git Branching Homework

## Описание проекта

Учебный проект для практики работы с Git и GitHub.
В проекте были использованы отдельные ветки для разработки новых функций, их слияние через Merge и Pull Request.

## Использованные ветки

| Ветка | Что делал | Как попала в main |
|---------|---------|---------|
| feature/profile | модуль профиля студента | merge локально |
| feature/subjects | список дисциплин | merge локально |
| feature/report | итоговый отчёт | Pull Request |
| experiment/broken-idea | тестовая идея | удалена без merge |

## Коммиты

1. docs: добавить README проекта
2. feat: добавить базовый запуск приложения
3. feat(profile): создать профиль студента
4. docs(profile): обновить README
5. feat(subjects): добавить список дисциплин
6. refactor(main): подключить subjects
7. feat(report): добавить генерацию отчёта
8. refactor(main): подключить отчёт
9. chore: удалить экспериментальную ветку



## Вывод

В ходе работы я научился создавать отдельные ветки для разработки новых функций, выполнять их слияние через Merge и Pull Request, а также поддерживать чистую историю изменений в Git.
