# Git Branching Homework

## Описание проекта

Учебный проект для практики работы с Git и GitHub.
В проекте были использованы отдельные ветки для разработки новых функций, их слияние через Merge и Pull Request.

## Использованные ветки

| Ветка | Что делал | Как попала в main |
|---------|---------|---------|
| feature/profile | модуль профиля студента | merge локально |
| feature/subjects | список дисциплин | merge локально |
| feature/report | итоговый отчёт | Pull Request |
| experiment/broken-idea | тестовая идея | удалена без merge |

## Коммиты


docs: добавить README проекта

feat: добавить базовый запуск приложения

feat(profile): добавить модуль профиля студента

docs(profile): обновить README для модуля профиля

feat(subjects): добавить список дисциплин

refactor(main): подключить модуль subjects

feat(report): добавить генерацию итогового отчёта

refactor(main): подключить отчёт к приложению

docs: добавить скриншоты выполнения задания
